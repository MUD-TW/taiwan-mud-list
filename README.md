## 使用說明

```bash
# 下載專案
$ git clone https://github.com/MUD-TW/taiwan-mud-list.git
$ git checkout nodejs

# 安裝依賴
$ npm install

# 更新列表
$ npm run update tw
$ npm run update cn
```
